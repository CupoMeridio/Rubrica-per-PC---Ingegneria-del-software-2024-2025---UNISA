Descrizione dettagliata

La classe RunMain funge da punto di ingresso principale per l'applicazione.

Questa classe è stata creata esclusivamente per far funzionare correttamente il file JAR del progetto JavaFX. Senza di essa, le dipendenze dell'applicazione non venivano riconosciute correttamente.

Il metodo main(String[]) invoca il metodo App#main(String[]) per avviare l'applicazione JavaFX, gestendo correttamente le dipendenze necessarie al funzionamento dell'applicazione.
